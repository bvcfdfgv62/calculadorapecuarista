# 🐄 Calculadora Pecuária

Sistema desenvolvido para cálculo de custos e análise de lucro na produção pecuária.

---

## 💡 Sobre o projeto

Aplicação voltada para análise financeira no setor agro, permitindo simular custos, receitas e margem de lucro.

---

## ⚙️ Tecnologias

- React + TypeScript
- Vite
- Integração com APIs

---

## 📊 Funcionalidades

- Cálculo de custo por animal
- Estimativa de lucro
- Simulação de cenários
- Interface simples e rápida

---

## ⚠️ Observação

Este projeto é uma versão demonstrativa baseada em experiências no setor agro.  
Não contém dados reais ou informações confidenciais.
